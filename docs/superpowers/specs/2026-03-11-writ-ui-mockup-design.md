# Writ UI 목업 디자인 스펙

## 개요

Writ은 구독 없이 온디바이스에서 동작하는 음성 전사 앱이다. 이 문서는 전체 플랫폼(iPhone, iPad, Mac, Apple Watch)의 UI 디자인을 정의한다.

**디자인 원칙:**
- Apple HIG 준수 (SF Pro, 시스템 컬러, 표준 컴포넌트)
- 녹음 버튼은 항상 눈에 띄고 접근성 높게
- 상태 피드백이 명확하게 (녹음 중, 전사 중, 완료)
- 라이트 모드 우선, 녹음 중에는 다크 배경으로 전환

**목업 파일 위치:** `.superpowers/brainstorm/20316-1773210238/`

## 네비게이션 플로우

### iPhone/iPad 메인 플로우

```
[녹음 탭] ──녹음 완료──▶ [기록 탭] (자동 전환, 새 항목 상단)
                              │
                         항목 탭 ▼
                        [전사 상세]
                              │
                     ··· 메뉴 ▼
              [재전사 모델 선택 시트]
```

### 녹음 → 전사 → 결과 플로우

```
녹음 시작 → 녹음 중 (DI 활성) → 녹음 중지
  → 전사 시작 (기록 탭에 "전사 중 N%" 표시)
  → 전사 완료 → 알림 배너 ("전사 완료 — 탭하여 확인")
  → [자동 복사 ON이면] 클립보드 복사 + 토스트 ("클립보드에 복사됨")
```

### 모델 폴백 플로우

```
large 모델 전사 시작 → 메모리 부족/실패
  → 인라인 배너: "large-v3 전사에 실패했습니다. small 모델로 다시 전사합니다."
  → small 모델로 자동 재전사
```

### Mac fn 키 빠른 녹음 플로우

```
fn 꾹 누르기 → 메뉴바 아이콘 빨간 펄스 + 녹음 시작
  → fn 놓기 → 녹음 중지 + 전사 시작
  → 메뉴바 스피너: "전사 중..."
  → 전사 완료 → 클립보드 자동 복사 → 포커스된 인풋에 자동 붙여넣기
  → 메뉴바 체크마크(1초) → 원래 아이콘 복귀
```

## 공통 상태 UI

### 빈 상태 (Empty State)

**기록 탭 빈 상태:**
- 중앙에 마이크 아이콘 (회색, 48px)
- "아직 녹음이 없습니다"
- "녹음 탭에서 첫 녹음을 시작해보세요" (서브텍스트)

**Mac 사이드바 빈 상태:**
- 동일한 패턴, 사이드바 중앙에 표시

### 알림/피드백 UI

**전사 완료 알림:**
- iOS 시스템 알림 배너: "Writ — 전사 완료" + 녹음 제목
- 앱 내: 기록 탭 상단에 인라인 배너 (초록 배경, "전사 완료 — 탭하여 확인", 3초 후 자동 사라짐)

**클립보드 복사 토스트:**
- 화면 상단 드롭다운 토스트: "📋 클립보드에 복사됨" (2초 표시)

**모델 폴백 배너:**
- 전사 상세 화면 상단에 주황 인라인 배너: "⚠️ large-v3 전사에 실패하여 small 모델로 재전사했습니다"
- 닫기(×) 버튼 포함

**iCloud 용량 경고:**
- 설정 탭 저장 공간 섹션: 사용량이 90% 초과 시 빨간 텍스트 "iCloud 저장 공간이 부족합니다. 오래된 녹음을 삭제하거나 자동삭제를 설정하세요."

### 최초 실행 (온보딩)

- 별도 온보딩 화면 없이, 최초 실행 시 기기에 맞는 기본 모델을 자동 다운로드
- 설정 탭에 다운로드 프로그레스 표시
- 녹음 탭에 모델 준비 상태 인디케이터: "모델 다운로드 중..." → "● small 모델 사용 중"

## iPhone (4개 핵심 화면 + 2개 특수 UI)

### 1. 녹음 탭 (`iphone-record.html`)

**구조:** 3-탭 구조 (녹음, 기록, 설정)

**대기 상태:**
- 라이트 배경 (`#f2f2f7`)
- Large Title 네비게이션: "녹음"
- 언어 선택 칩 (예: "🌐 한국어 ▼") — 상단 중앙
- 정적 오디오 파형 바 — 중앙 영역
- 타이머 표시: `00:00.00` (얇은 폰트)
- 힌트 텍스트: "탭하여 녹음 시작"
- 빨간 원형 녹음 버튼 (80px, 4px 링 + 60px 내부 원)
- 모델 인디케이터: "● small 모델 사용 중"

**녹음 중 상태:**
- 배경이 다크(#000)로 전환 (0.3s ease-in-out)
- 파형 바에 애니메이션 적용
- 타이머 실시간 업데이트
- 녹음 버튼 내부가 네모(정지 모양)로 변환
- "● 녹음 중" 빨간색 라벨
- Dynamic Island 활성화

**녹음 완료 후:** 자동으로 기록 탭으로 전환, 새 항목이 최상단에 "전사 중" 상태로 표시

### 2. 기록 탭 (`iphone-history.html`)

- Large Title: "기록"
- 검색바 (iOS 표준)
- 날짜별 그룹 섹션 (오늘/어제/날짜)
- 각 항목 구성:
  - 상태 아이콘 (체크=완료, 스피너=진행중, 시계=대기)
  - 제목
  - 전사문 미리보기 (1줄, 완료된 항목만)
  - 메타: 녹음 길이 + 전사 상태 뱃지
  - 진행 중인 항목: 프로그레스바 표시 (예: "전사 중 67%")
- 스와이프 액션: 공유(파란), 삭제(빨간)
- **빈 상태:** 마이크 아이콘 + "아직 녹음이 없습니다"

### 3. 전사 상세 화면 (`iphone-detail.html`)

- 내비게이션: "‹ 기록" + ··· 더보기 메뉴
- **··· 메뉴 항목:** 다른 모델로 재전사, 이름 변경, 전사문 복사, 공유, 삭제
- 제목 + 메타 (날짜, 길이, 사용 모델)
- 오디오 플레이어:
  - 파형 시크바 (재생된 부분=파란, 미재생=회색)
  - 현재 시간 / 남은 시간
  - 재생/일시정지, 스킵(앞/뒤), 배속 버튼 (0.5x, 1.0x, 1.5x, 2.0x)
- 액션 바: 복사, 공유, 내보내기
- 타임스탬프별 전사문 세그먼트
  - 현재 재생 중인 세그먼트 하이라이트 (파란 왼쪽 보더)
  - 세그먼트 탭 → 해당 오디오 위치로 이동
- **내보내기 시트:** TXT (순수 텍스트), SRT (타임스탬프 자막), 클립보드 복사 — 바텀 시트로 표시
- **재전사 플로우:**
  1. ··· 메뉴 → "다른 모델로 재전사" 탭
  2. 모델 선택 바텀 시트 표시 (현재 모델 체크마크, 미다운로드 모델은 "다운로드" 버튼)
  3. large 모델 선택 시 경고: "전사 시간이 더 걸리며, 다른 앱 사용 시 중단될 수 있습니다"
  4. "재전사" 버튼 탭 → 전사 상세 화면에서 프로그레스 표시
  5. 완료 시 전사문 갱신 + 토스트 "재전사 완료"
  6. 실패 시 인라인 배너로 폴백 안내

### 4. 설정 탭 (`iphone-settings.html`)

- Large Title: "설정"
- **전사 모델 섹션:**
  - 5개 모델 목록 (tiny, base, small, large-v3, large-v3 turbo)
  - 각 모델: 이름, 디스크 크기, 특징 설명
  - 사용 중 모델: 파란 "사용 중" 뱃지
  - 다운로드 필요 모델: "다운로드" 버튼
  - 다운로드 중: 스피너 + 퍼센트
  - 미지원 모델 (구형 기기): 비활성(opacity 0.4) + 빨간 사유 안내 ("이 기기의 RAM이 부족하여 사용할 수 없습니다")
  - 섹션 푸터: "기기에서 지원하는 모델만 표시됩니다."
- **언어:** 전사 언어 선택 (Whisper 지원 다국어)
- **동작:** 전사 후 자동 복사 토글, iCloud 동기화 토글
- **저장 공간:** iCloud 사용량 바 (90% 초과 시 경고), 자동 삭제 기간 (30/90/180일/삭제 안 함)
- **정보:** 버전

### 5. Dynamic Island (`iphone-special.html`)

- **Compact:** 200px, 빨간 점(펄스) + 시간 + 미니 파형
- **Expanded:** 360px, 앱 이름 + 큰 타이머 + 파형 + 정지 버튼 (둥근 네모)
- **잠금 화면 Live Activity:** 앱 아이콘(W, 그라데이션) + "Writ 녹음 중" + 시간 + 정지 버튼

### 6. 커스텀 키보드 (`iphone-special.html`)

**키보드 높이:** 260px (iOS 표준 키보드와 동일)

**대기:**
- 헤더: Writ 로고(W, 20px) + 지구본 키(키보드 전환)
- 큰 빨간 마이크 버튼 (64px)
- "탭하여 음성 입력" 힌트
- "📋 최근 전사문 삽입" 버튼 (파란 칩)

**녹음 중:**
- 헤더 다크 전환
- 다크 배경 (#1c1c1e)
- 파형(14개 바) + 타이머(24px) + "● 녹음 중" + 정지 버튼(48px 링 + 20px 네모)

**전사 중:**
- 스피너 + "전사 중..." (메인 앱에 전사 위임, App Group으로 결과 전달)

**삽입 완료:**
- 초록 체크마크(48px) + "전사 완료 — 텍스트 삽입됨"
- 전사된 텍스트가 `textDocumentProxy.insertText()`로 입력 필드에 자동 삽입
- "다시 녹음" 버튼 (작은 마이크, 48px)

**보안 필드:** 비밀번호/전화번호 필드에서는 시스템 키보드로 자동 전환

## Mac (2개 화면)

### 7. 메뉴바 드롭다운 (`mac-ui.html`)

**대기:**
- 메뉴바 아이콘: Writ 로고 (16px, 그라데이션)
- 드롭다운 (320px):
  - 녹음 시작 버튼 (빨간, 전체 너비)
  - fn 키 단축키 안내: `fn 길게 누르기로 빠른 녹음`
  - 현재 모델 표시 (예: "● large-v3 turbo")
  - 최근 전사 목록 (3개, 각 항목에 📋 클립보드 복사 아이콘)
  - "메인 윈도우 열기" 링크

**녹음 중:**
- 메뉴바 아이콘 빨간 펄스
- 드롭다운: 다크 녹음 버튼 (빨간 점 펄스 + 시간 + "■ 중지")
- `fn 놓기로 녹음 중지 + 전사 + 붙여넣기` 안내

**fn 키 빠른 녹음 후 피드백:**
- 녹음 중지 → 메뉴바 아이콘에 스피너 표시 + "전사 중..."
- 전사 완료 → 클립보드 복사 → 포커스된 인풋에 ⌘V 자동 붙여넣기
- 메뉴바 아이콘에 체크마크(✓) 1초 표시 후 원래 아이콘 복귀
- 실패 시: macOS 알림 "전사에 실패했습니다"

### 8. 메인 윈도우 (`mac-ui.html`)

- macOS 표준 윈도우 (트래픽 라이트, 타이틀바 "Writ")
- 최소 윈도우 크기: 800×500px
- **툴바:** 녹음 버튼(빨간), 모델 선택 드롭다운, 설정 아이콘(⚙️)
- **사이드바 (260px, 리사이즈 가능):**
  - 검색바
  - 날짜별 녹음 목록
  - 상태 점 (녹색=완료, 주황=진행중, 회색=대기)
  - 선택된 항목: 파란 하이라이트
  - 빈 상태: 마이크 아이콘 + "녹음이 없습니다"
- **메인 영역:**
  - 제목(22px, 600) + 메타 (날짜, 길이, 사용 모델)
  - 오디오 플레이어 (재생 버튼, 시크바, 시간, 배속)
  - 액션 바: 📋 복사, 📤 공유, 📄 TXT, 🎬 SRT — 내보내기 시 macOS 저장 다이얼로그
  - 타임스탬프별 전사문 (현재 재생 세그먼트 하이라이트)
- **설정 (⚙️ 클릭 시 시트 또는 별도 윈도우):**
  - 모델 관리: iPhone 설정 탭과 동일한 모델 목록/다운로드 UI
  - 전사 언어 선택
  - 전사 후 자동 복사 토글
  - 자동 삭제 기간
  - fn 키 단축키 활성화/비활성화
  - 글로벌 단축키 설정
  - (v2) Mac 서버 모드 ON/OFF (비활성 + "v2에서 제공 예정")

## iPad

### 9. iPad 메인 화면 (`ipad-watch.html`)

iPhone과 동일한 3-탭 구조. 넓은 화면을 활용한 적응형 레이아웃.

**가로 모드 — Split View:**
- **사이드바 (340px):** Large Title "기록", 검색바, 날짜별 녹음 목록 (iPhone 기록 탭과 동일한 항목 구성)
- **메인 영역:** 전사 상세와 동일 (제목, 플레이어, 액션, 전사문)
- **탭바:** 중앙 정렬 (녹음, 기록, 설정) — 아이콘+라벨

**세로 모드:**
- iPhone과 동일한 단일 컬럼 레이아웃
- 기록 탭에서 항목 탭 시 전사 상세로 전체 화면 전환

**녹음 탭:** iPhone 녹음 탭과 동일한 레이아웃 (중앙 녹음 버튼, 파형, 타이머). 넓은 화면에서 파형이 더 길게 표시.

**설정 탭:** iPhone 설정 탭과 동일한 구성. 가로 모드에서 설정 그룹이 좌우 2열로 배치될 수 있음.

**커스텀 키보드:** iPhone과 동일한 구조, 키보드 높이가 iPad에 맞게 조정 (약 300px).
